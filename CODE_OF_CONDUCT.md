# Code of Conduct

This project is a technical, research-oriented open-source repository. Contributors are expected to keep discussions specific, evidence-based, and respectful.

## Expected behavior

- Keep reviews focused on source quality, tests, documentation, and reproducibility.
- Challenge claims that are unsupported by artifacts or tests.
- Keep market interpretation monitor-only and avoid financial-advice language.

## Unacceptable behavior

- Harassment, abuse, or personal attacks.
- Promoting guaranteed profitability, order execution, or trade-call behavior through this project.
- Submitting private broker credentials, paid vendor data, or personally identifiable information.

## Enforcement

Maintainers may close issues, reject pull requests, or block users who repeatedly violate this code of conduct.
